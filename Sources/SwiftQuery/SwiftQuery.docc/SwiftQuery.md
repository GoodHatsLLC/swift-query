# SwiftQuery

SwiftQuery is a React Query-inspired data fetching library for SwiftUI with tag-based cache invalidation and persistent storage.

## Overview

- Declarative data fetching via ``Query``.
- Mutations with automatic invalidation via ``Mutation``.
- Hierarchical invalidation via ``QueryTag``.
- Cache coordination via ``QueryClient``.
